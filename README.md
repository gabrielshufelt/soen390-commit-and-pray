# SOEN 390 – Commit and Pray  
## Campus Navigation Application for Concordia University

Navigate Concordia’s campuses with ease.

**Commit and Pray** is a mobile application designed to help students, staff, and visitors efficiently navigate both **SGW and Loyola campuses**, indoors and outdoors. The app provides **building search, directions, and accessibility-aware routing**, making it easier to get where you need to go.

Built as part of the **SOEN 390 Software Engineering Project**, this application combines real-world navigation challenges with modern mobile development practices.

---

## Key Features

- Interactive campus maps (SGW & Loyola)  
- Building search and information  
- Directions across and between campuses  
- Accessibility-aware navigation  
- Indoor and outdoor routing  
- Points of interest (elevators, washrooms, food, etc.)
